# Diachronic Sample Integration

This repository accompanies the paper:

**“Diachronic Sample Integration: Robust Tail-Risk Estimation with Generative Models”**

Submitted to **ICDM 2026 Applied Track**.

## Overview

This repository contains the appendix for the paper. The appendix provides additional details, proofs, implementation notes, and experimental results that were omitted from the main paper due to space constraints.

## Repository Contents

* [`appendix.pdf`](./appendix.pdf)
  Supplementary appendix containing additional methodological details, theoretical discussion, proofs, and extended experimental results.

## Code Availability

The full code for all experiments in the paper will be made publicly available upon acceptance.
